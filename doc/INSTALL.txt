Dependencies
============

apt-get install python3-dbus

openheating
===========

* Package installation

  python3 setup.py sdist --formats=bztar
  python3 setup.py install --prefix=/usr --install-layout=deb

* Setup init scripts

  Symlink to /etc/init.d/ from /usr/share/openheating/debian/ as needed.

* Setup config files

  Copy to /etc/openheating/ from /usr/share/openheating/debian/ as
  needed, and edit.

* Add user and group

  groupadd openheating
  useradd --home-dir /var/run/openheating --gid openheating --no-create-home --shell /bin/false openheating

* Enable and start services

  update-rc.d openheating-sensors defaults
